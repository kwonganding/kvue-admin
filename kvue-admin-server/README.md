
# kvue-admin-server

基于NodeJS + Express 构建的后端服务，实现的比较简单，主要为了满足前端框架基本需求。

---

# 接口规范协议
```
{
  code:0,        // 状态码
  data:{},       // 数据
  mes:'消息'      // 错误消息或描述信息
}
```
**code 编码：**

- 0：默认值，表示正常
- 4001：授权token无效，请重新登录
- 4002：授权token过期
- 其他：都为异常错误
